# VGA-Automator
floorplan-to-dxf converter

## PyQt6 데스크톱 앱

VGA Automator는 드래그 앤 드롭을 지원하는 PyQt6 데스크톱 GUI를 제공합니다.

### 핵심 기능

1. **드래그 앤 드롭**: 도면 이미지를 앱에 드래그하여 로드
2. **실시간 미리보기**: 업로드된 도면 미리보기
3. **설정 패널**: 감지 신뢰도, 갭 크기, 스케일 조절
4. **진행 상태 표시**: 변환 진행률 및 상태 메시지
5. **백그라운드 처리**: QThread를 사용한 비동기 변환 (UI 멈춤 방지)
6. **결과 저장**: DXF 파일 저장 다이얼로그

### 앱 특징

1. **드래그 앤 드롭**: 파일을 앱 창에 드래그하면 자동 로드
2. **미리보기**: 도면 이미지 미리보기 표시
3. **설정 패널**: 
   - 감지 신뢰도 슬라이더
   - 갭 연결 크기 조절
   - 출력 스케일 설정
4. **진행 상태**: 프로그레스 바 + 단계별 메시지
5. **결과 통계**: 벽체/문/창문 개수 표시
6. **파일 탐색기 열기**: 변환 완료 후 저장 폴더 열기 옵션
7. **비동기 처리**: UI 멈춤 없이 백그라운드 변환

### 사용법

```bash
# 의존성 설치
pip install -r requirements.txt

# 또는 PyQt6만 설치
pip install PyQt6

# 앱 실행
python app_desktop.py
```

### 요구사항

- Python 3.8 이상
- PyQt6 6.5.0 이상
- opencv-python, numpy, ezdxf 등 (requirements.txt 참조)
